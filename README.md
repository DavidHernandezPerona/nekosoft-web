# Nekosoft

Nekosoft es un proyecto de desarrollo web creado con el objetivo de aprender y poner en práctica tecnologías web modernas desde cero.

La aplicación consiste en una página web corporativa desarrollada con HTML5, CSS3 y JavaScript, incorporando diseño responsive, modo oscuro y funcionalidades dinámicas para mejorar la experiencia de usuario.

## Demo online

https://davidhernandezperona.github.io/nekosoft-web/

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* GitHub Pages

## Funcionalidades

* Navegación multipágina.
* Diseño responsive para dispositivos móviles.
* Sección de servicios.
* Sección de proyectos.
* Formulario de contacto con validación.
* Mensajes de error dinámicos.
* Contador de caracteres en el formulario.
* Mensaje de confirmación de envío.
* Modo oscuro persistente mediante Local Storage.
* Cambio automático de logotipo según el tema seleccionado.
* Animaciones visuales y efectos hover.

## Objetivos del proyecto

* Aprender desarrollo web paso a paso.
* Aplicar buenas prácticas de maquetación y estilos.
* Introducir programación con JavaScript.
* Utilizar Git y GitHub para control de versiones.
* Crear un proyecto real para portfolio profesional.
