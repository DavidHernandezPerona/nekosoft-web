# Nekosoft

Nekosoft es un proyecto de desarrollo web creado con el objetivo de aprender y poner en práctica tecnologías web modernas desde cero.

Este proyecto se desarrollará progresivamente, comenzando con HTML5 y ampliándose posteriormente con CSS3 y JavaScript para construir una página web corporativa completa y profesional.

Durante el desarrollo se aplicarán buenas prácticas de programación, organización de proyectos y control de versiones mediante Git y GitHub.

## Tecnologías

- HTML5
- CSS3 (próximamente)
- JavaScript (próximamente)

## Objetivos del proyecto

- Aprender desarrollo web paso a paso.
- Crear una página web profesional desde cero.
- Aplicar buenas prácticas de organización y mantenimiento del código.
- Utilizar Git y GitHub como herramientas de control de versiones.
- Construir un proyecto que sirva como portfolio de aprendizaje y evolución profesional.

## Estado del proyecto

Actualmente en fase inicial de desarrollo.
