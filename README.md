# Nekosoft

Nekosoft es un proyecto de desarrollo web creado con el objetivo de aprender y poner en práctica tecnologías web modernas desde cero.

El proyecto consiste en el desarrollo de una página web corporativa, aplicando progresivamente HTML5, CSS3 y JavaScript para construir una solución funcional, visualmente atractiva y bien estructurada.

Durante el desarrollo se aplican buenas prácticas de programación, organización de proyectos y control de versiones mediante Git y GitHub.

## Tecnologías

- HTML5
- CSS3
- JavaScript (en desarrollo)

## Objetivos del proyecto

- Aprender desarrollo web paso a paso.
- Crear una página web profesional desde cero.
- Aplicar buenas prácticas de organización y mantenimiento del código.
- Utilizar Git y GitHub como herramientas de control de versiones.
- Construir un proyecto que sirva como portfolio de aprendizaje y evolución profesional.

## Estado del proyecto

Actualmente cuenta con:

- Estructura completa en HTML.
- Diseño y estilos desarrollados con CSS.
- Navegación multipágina.
- Secciones de servicios, proyectos y contacto.
- Formulario de contacto.
- Uso de Git y GitHub para el control de versiones.

Próxima fase:

- Implementación de funcionalidades con JavaScript.
